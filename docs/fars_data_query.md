# Data Querying Steps

Start [here](https://www-fars.nhtsa.dot.gov//QueryTool/QuerySection/SelectYear.aspx) for downloading data.  For the years 2015 and 2016:

1. Select Option 1:
   - Atmospheric Condition (1 & 2)
   - City
   - County
   - Crash Date
   - Crash Time
   - All lat/long
   - Age
   - Alcohol Test Results
   - Alcohol Test Status
   - Injury Severity 
   - Person Type
   - Sex
